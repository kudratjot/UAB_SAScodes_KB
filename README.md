## Overview
This repository contains SAS code samples from clinical research projects conducted at UAB's Department of Internal Medicine & Population Science.
Each script demonstrates a complete analytical pipeline — from data import and cleaning through statistical modeling and output generation.
> **Note:** All data files have been excluded. File paths have been generalized. Patient identifiers have been removed or anonymized.
>  These scripts are shared to demonstrate statistical methodology only.


## SAS Procedures & Techniques

- **Regression:** PROC LOGISTIC, PROC REG, PROC GLM
- **Survival Analysis:** PROC LIFETEST (Kaplan-Meier), PROC LOGISTIC with ROC
- **Nonparametric Tests:** PROC NPAR1WAY (Wilcoxon), PROC UNIVARIATE (signed-rank)
- **Agreement & Reliability:** Kappa statistics, ICC, Bland-Altman
- **Effect Size:** Cohen's d (manual calculation), ROC AUC, Youden's index
- **Descriptive & Exploratory:** PROC MEANS, PROC FREQ, PROC CORR (Spearman/Pearson)
- **Data Management:** PROC IMPORT, DATA step (derived variables, categorization, missing imputation), PROC TRANSPOSE, ODS Excel output
- **Visualization:** PROC SGPLOT (bar charts, grouped displays)
